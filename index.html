ABOUT ME

hi! i'm rafaela cristina and i'm a 2D artist from brasil. i'm currently working on building this portfolio and i hope to improve it soon! here you can find two games that i developed for a graduation class: human invasion and physicat. the former is a simple 2D space-shooter, and the latter is an educational game about basic physics concepts. the programming for level 2 was done by a classmate :)
the last one is one of my personal projects: "animal forest". i intend to recreate the art style of animal crossing games in a retro style!
i also enjoy game design and storytelling. i'm currently working on concept art for a fake 2D mobile game and some other fanart.
